A component is one file

Markup + <script> + <style>. Every let is reactive — reassign it and the DOM updates. 

<h2>{count}</h2>
<button onclick="count++">+1</button>

<script>
  let count = 0;
</script>

<style>
  h2 { color: #ff6b35; }
</style>

Interpolation

Braces take any JS expression.

<p>{user.name}</p>
<p>{items.length} items</p>
<p>{age >= 18 ? 'adult' : 'minor'}</p>

<!-- works in attributes too -->
<li class="{item.done ? 'done' : ''}">

Reactivity gotcha: arrays & objects

Reassign — don't mutate in place.

items = [...items, newItem];        // ✅
items = items.filter(i => i.id !== id); // ✅

Conditionals

<if condition="status === 'loading'">⏳</if>
<else-if condition="status === 'error'">💥</else-if>
<else>✅</else>

<!-- toggle CSS display, keep the DOM -->
<show condition="open">…</show>

Lists

<for each="fruit in fruits">
  <li>{fruit}</li>
</for>

<!-- with index + key -->
<for each="(item, i) in items" key="item.id">
  <li>#{i + 1} — {item.name}</li>
</for>

Events

Plain onclick etc. run in component scope. Use $on: for modifiers. 

<button onclick="count++">+</button>
<button onclick="save(item.id)">Save</button>

<form $on:submit.prevent="add()">…</form>
<input $on:keyup.enter="search()" />
<div $on:click.self="closeModal()">…</div>

Modifiers: .prevent .stop .self .once .capture .passive .enter .escape .ctrl .shift .alt .meta

Two-way binding

<input $bind="name" />
<textarea $bind="bio"></textarea>
<select $bind="color">…</select>
<input type="range" $bind="size" />

Element refs

<input $ref="inputEl" />

<script>
  function focusIt() {
    $refs.inputEl.focus();
  }
</script>

Props (parent → child)

Attributes on the tag override the child's let defaults. 

<!-- parent -->
<user-card name="{person.name}" role="Engineer">
</user-card>

<!-- child (user-card) -->
<strong>{name}</strong> <em>{role}</em>
<script>
  let name = "Unknown"; // overridden by attribute
  let role = "—";
</script>

Event bus (any component ↔ any component)

// sender
$emit("chat:message", { text: "hi" });

// receiver
$listen("chat:message", (data) => {
  messages = [...messages, data];
});

Literal braces

$no:bind skips interpolation inside an element (this cheat sheet uses it!). 

<code $no:bind>{not interpolated}</code>

Registering components

How the playground wires files to tags — new tabs you add on the left are registered the same way.

import { registerComponents } from "ladrillosjs";

await registerComponents([
  { name: "user-card", path: "./user-card.html" },
]);
// then use <user-card></user-card> anywhere
